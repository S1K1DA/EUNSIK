정리
  1. font-size : 텍스트의 크기를 결정(px, %, em, rem, vh...)
  2. text-align : 텍스트의 정렬 방식을 결정
    • left : 왼쪽 정렬
    • center : 중앙 정렬
    • right : 오른쪽 정렬
  3. font-weigt : 텍스트의 굵기를 결정
    • normal : 일반 텍스트(기본값)
    • bold : 굵은 텍스트
    • 100~900 : 값이 클수록 텍스트가 굵어짐
  4. font-style : 텍스트의 기울기를 결정
    • normal : 일반 텍스트(기본값)
    • italic : 기울어진 텍스트
    • oblique : 기울어진 텍스트(일부 글꼴에서는 bblique로 지원)
  5. text-decoration : 텍스트에 밑줄, 취소선, 윗줄 등의 효과를 지정
    • none : 일반 텍스트(기본값)
    • underline : 밑줄
    • line-through : 취소선
    • overline : 윗줄
    • underline overline : 밑줄+윗줄
  6. text-shadow : 텍스트에 그림자 효과를 지정
